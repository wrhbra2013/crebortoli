Ativar Servidor PHP

1. Verificar PHP

php --version

Se não tiver, instale: sudo apt install php php-sqlite3

2. Iniciar Servidor

Na pasta do projeto:

php -S localhost:8080

Ou para disponibilizar na rede:

php -S 0.0.0.0:8080

3. Testar API

Acesse no navegador:

Testar API

Ou via terminal:

curl http://localhost:8080/api.php

Após ativar:

  • O database.sqlite será criado automaticamente
  • Os dados sincronizam entre dispositivos
  • O login QR funcionará

4. Hospedagem

Para colocar online, basta um servidor com PHP:

  • HostGator, LocaWeb, Hostinger
  • Vercel (com PHP adapter)
  • Railway, Render (PHP não suportado nativamente)
