Google Forms para Escrita de Dados

Resumo: O sistema lê do Google Sheets, mas para ESCREVER dados, usamos Google Forms.

Formulários Necessários

DadosFormulárioPara...
AgendamentosGoogle Form 1Cliente agendar
ContatosGoogle Form 2Cliente enviar mensagem
ClientesGoogle Form 3Cadastro opcional

Passo a Passo

1. Criar Google Form (Agendamentos)

  • Acesse forms.google.com
  • Crie novo formulário
  • Nome: "Agendamento - Cre Bortoli"
  • Campos necessários:
    • Data (date)
    • Horário (dropdown)
    • Nome (text)
    • WhatsApp (text)
    • Serviço (dropdown)
  • Configurações → Permitir Apenas uma resposta
  • Obter link de compartilhamento

2. Criar Google Form (Contato)

  • Crie novo formulário
  • Nome: "Contato - Cre Bortoli"
  • Campos:
    • Nome (text)
    • Email (email)
    • Mensagem (paragraph)

3. Configurar no Código

Após criar os forms, você terá URLs como:

https://docs.google.com/forms/d/e/SEU_FORM_ID_1/viewform
https://docs.google.com/forms/d/e/SEU_FORM_ID_2/viewform

Configure no sistema em static/js/storage.js

Como Funciona

┌─────────────────┐     ┌──────────────────┐
│  Cliente        │     │  Google Sheets  │
│  (navegador)    │     │  (LEITURA)       │
└────────┬────────┘     └──────────────────┘
         │
         │ submit → Google Form
         ▼
┌──────────────────────────────────┐
│  Google Form                     │
│  (abre no navegador do cliente)  │
└──────────────────────────────────┘
         │
         │ resposta
         ▼
┌──────────────────────────────────┐
│  Planilha Google Sheets         │
│  (dados inseridos automaticamente)│
└──────────────────────────────────┘

Fluxo Recomendado

  1. Leitura: Dados carregam do Google Sheets
  2. Escrita: Cliente clica → Abre Google Form → Preenche → Envia
  3. Sync: Ao recarregar, dados atualizam na planilha
Dica: No formulário de agendamento, configure "Respostas → Conectar com planilha" para os dados irem direto para a planilha compartilhada.
