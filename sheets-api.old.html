Google Sheets como API

1. Criar Planilha

  • Acesse Google Sheets
  • Crie uma nova planilha
  • Nomeie como crebortoli_db

2. Criar Abas

Crie as abas (sheets) com nomes:

  • agendamentos
  • servicos
  • clientes
  • receitas
  • contatos

3. Tornar Público

  • Clique em Compartilhar
  • Clique em Compartilhar com pessoas
  • Defina como Público na web (Qualquer pessoa pode visualizar)

4. Obter ID

Na URL da planilha:

https://docs.google.com/spreadsheets/d/ABC123DEF456GHI789/edit

O ID é: ABC123DEF456GHI789

5. Testar API

URL da API (formato JSON):

https://spreadsheets.google.com/feeds/list/SEU_ID/od6/public/values?alt=json

Ou formato CSV (mais simples):

https://docs.google.com/spreadsheets/d/SEU_ID/export?format=csv&sheet=agendamentos

6. No Código

// Ler dados
const response = await fetch('https://docs.google.com/spreadsheets/d/SEU_ID/export?format=csv');
const texto = await response.text();
