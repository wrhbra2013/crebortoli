Setup PocketBase

Configure as collections necessárias para o sistema.

1. Acesse o Admin do PocketBase

Vá até: https://www.crebortoli.com.br/_/

2. Crie as Collections

No painel admin, crie cada collection com os campos abaixo:

agendamentos

Campos:
- data (date)
- hora (text)
- cliente (text)
- telefone (text)
- servico (text)
- servicoNome (text)
- valor (number)
- status (select: PENDENTE, PAGO, CONCLUIDO)
- pago (bool)

servicos

Campos:
- nome (text)
- preco (number)
- categoria (text)
- ativo (bool)

clientes

Campos:
- nome (text)
- telefone (text)
- email (email)
- anamnese (json)

receitas

Campos:
- nome (text)
- produtos (json)
- instrucoes (text)

qr_sessoes

Campos:
- token (text)
- status (select: aguardando, aprovado, negado, expirado)
- urlAprovacao (url)
- created (date)
- aprovadoEm (date)
- negadoEm (date)

3. Configure API Rules

Para cada collection, defina as regras de API (em "Settings" > "API rules"):

List/View:

@request.auth.id != ""

Create:

@request.auth.id != ""

Update:

@request.auth.id != ""

Delete:

@request.auth.id != ""

4. Criar Usuário Admin

Em "Collections" > "users", adicione usuários em "Admin panel".
